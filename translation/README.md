# 翻訳機能

- Python 3.10.9
- pip 23.1

## 環境構築

- 仮想環境の用意

```bash
python3 -m venv .venv
```

- 仮想環境の起動

```shell
source .venv/bin/activate
```

- 必要なモジュールのインストール

```shell
pip install -r requirements.txt
```

- メモ
  - requirements.txtの作り方

	```shell
	pip freeze > requirements.txt
	```
